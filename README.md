## 🚀 快速开始

### 本地开发

```bash
cd personal-homepage
npm install
npm run dev
```

### 部署到 GitHub Pages

1. 创建名为 `sunjieseu.github.io` 的仓库
2. 推送代码到 main 分支
3. 在仓库设置中启用 GitHub Pages (选择 GitHub Actions)
